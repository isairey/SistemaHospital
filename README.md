## 📌 Descripción

**Danphe EMR** es un sistema empresarial basado en la web diseñado para la gestión hospitalaria integral.  
Actualmente es utilizado por **más de 60 hospitales** en países como:

- India 🇮🇳  
- Nepal 🇳🇵  
- Bangladesh 🇧🇩  
- Kenia 🇰🇪  

Ofrece una solución completa para operaciones médicas, administrativas y financieras.
